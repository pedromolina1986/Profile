HTML Cheat Sheet

A quick reference guide for HTML tags and attributes

HTML Structure

Every HTML document follows this basic structure:

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Document Title</title>
</head>
<body>
  <h1>Heading</h1>
  <p>Your content here</p>
</body>
</html>

Common HTML Tags

Tag Description Example
<html> Defines the root of the HTML document. <html> </html>
<head> Contains metadata like title, links, and scripts. <head> </head>
<title> Defines the title of the document (shown in the browser tab). <title>Page Title</title>
<body> Contains the content of the HTML document (text, images, etc.). <body> </body>
<p> Defines a paragraph of text. <p>This is a paragraph</p>
<a> Defines a hyperlink. <a href="https://example.com">Click here</a>
<img> Defines an image. <img src="image.jpg" alt="Description">
<ul> Defines an unordered list. <ul><li>Item 1</li></ul>
<ol> Defines an ordered list. <ol><li>Item 1</li></ol>
<li> Defines a list item (used inside <ul> or <ol>). <li>Item</li>
<div> Defines a division or section in an HTML document. <div>Content</div>
<span> Defines an inline container for text. <span style="color:red;">Red Text</span>
<table> Defines a table. <table>...</table>
<tr> Defines a table row. <tr>...</tr>
<td> Defines a table data/cell. <td>Data</td>
<th> Defines a table header. <th>Header</th>
<form> Defines an HTML form for user input. <form>...</form>
<input> Defines an input field. <input type="text" placeholder="Enter text">
<button> Defines a clickable button. <button>Click me</button>
<select> Defines a dropdown list. <select><option>Option 1</option></select>
<textarea> Defines a multi-line text input. <textarea>Enter text</textarea>
<header> Defines a header section. <header>Header Content</header>
<footer> Defines a footer section. <footer>Footer Content</footer>
<section> Defines a section of a document. <section>Section Content</section>
<article> Defines a self-contained article. <article>Article Content</article>
<aside> Defines content aside from the main content. <aside>Sidebar Content</aside>
<nav> Defines navigation links. <nav>...</nav>
<iframe> Embeds another webpage within the document. <iframe src="https://example.com"></iframe>

HTML Attributes

Attributes provide additional information about an HTML element. Some common attributes include:

  • id: Defines a unique identifier for an element.
  • class: Specifies a class for an element (can be used for styling).
  • src: Specifies the source for media elements like images or videos.
  • href: Specifies the URL for links.
  • alt: Provides an alternative text for images.

Form Elements

HTML forms are used to collect user input.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Semantic HTML

Semantic HTML uses tags that convey meaning, helping both browsers and developers.

  • <header>: Represents the header of a page or section.
  • <footer>: Represents the footer of a page or section.
  • <article>: Represents an independent piece of content.
  • <section>: Represents a section of content.
  • <aside>: Represents content related to the main content, such as a sidebar.
  • <main>: Represents the main content of the document.

Conclusion

This cheat sheet provides an overview of basic HTML tags and their usage. For more detailed learning, refer to HTML documentation and resources.

CSS Cheat Sheet

A quick reference guide for CSS properties and syntax

CSS Syntax

CSS syntax is simple and consists of a selector and a declaration block.

selector {
  property: value;
}

CSS Selectors

CSS selectors are used to target elements in your HTML document.

  • Universal selector: * - Targets all elements on the page.
  • Element selector: p - Targets all <p> elements.
  • Class selector: .class-name - Targets elements with a specific class.
  • ID selector: #id-name - Targets an element with a specific ID.
  • Attribute selector: [attribute] - Targets elements with a specific attribute.
  • Pseudo-classes: :hover, :focus, etc.
  • Pseudo-elements: ::before, ::after

Common CSS Properties

Property Description Example
color Sets the color of text. color: red;
background-color Sets the background color of an element. background-color: blue;
font-size Sets the size of the font. font-size: 16px;
margin Sets the margin around an element. margin: 10px;
padding Sets the padding inside an element. padding: 20px;
border Sets the border of an element. border: 1px solid black;
width Sets the width of an element. width: 100px;
height Sets the height of an element. height: 50px;
display Defines how an element is displayed. display: block;
font-family Sets the font family for text. font-family: Arial, sans-serif;

CSS Layout Properties

These properties control the layout and positioning of elements.

  • position: Controls the positioning of an element. Example: position: absolute;
  • top, right, bottom, left: Used to position elements when position is set to absolute, relative, or fixed.
  • float: Makes an element float to the left or right. Example: float: left;
  • clear: Clears floated elements. Example: clear: both;
  • flex: Flexbox layout for flexible and responsive designs. Example: display: flex;
  • grid: Grid layout for creating complex layouts. Example: display: grid;

CSS Color Values

Colors in CSS can be defined using names, hex codes, RGB, RGBA, HSL, or HSLA.

  • Named colors: Example: color: red;
  • Hex values: Example: color: #ff0000;
  • RGB values: Example: color: rgb(255, 0, 0);
  • RGBA values (with transparency): Example: color: rgba(255, 0, 0, 0.5);
  • HSL values: Example: color: hsl(0, 100%, 50%);
  • HSLA values (with transparency): Example: color: hsla(0, 100%, 50%, 0.5);

CSS Transitions and Animations

CSS transitions and animations allow you to create smooth changes between property values.

/* Transition example */
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
.box:hover {
  background-color: blue;
  transition: background-color 0.5s;
}

Conclusion

JavaScript Cheat Sheet

A quick reference guide for JavaScript syntax and concepts

JavaScript Syntax

JavaScript is a programming language used for creating interactive effects within web browsers.

The basic syntax includes variables, functions, conditionals, loops, and more.

Variables

In JavaScript, you can declare variables using let, const, or var.

  • let: Declares a block-scoped variable.
  • const: Declares a constant (cannot be reassigned).
  • var: Declares a function-scoped variable (not recommended due to scoping issues).
let name = "John";
const age = 30;
var isStudent = true;

Data Types

JavaScript supports the following data types:

  • String: Represents text. Example: "Hello, World!"
  • Number: Represents numeric values. Example: 123
  • Boolean: Represents true or false. Example: true, false
  • Array: Represents a list of values. Example: [1, 2, 3]
  • Object: Represents a collection of key-value pairs. Example: {name: "John", age: 30}
  • Null: Represents the absence of value. Example: null
  • Undefined: Represents an uninitialized variable.

Functions

Functions are reusable blocks of code that can be executed when called.

function greet(name) {
  return "Hello, " + name;
}

console.log(greet("John")); // Outputs: Hello, John

Conditionals

Conditionals allow you to execute code based on whether a condition is true or false.

if (age > 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

Loops

Loops are used to repeatedly execute a block of code.

1. For Loop

for (let i = 0; i < 5; i++) {
  console.log(i);
}

2. While Loop

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

3. For...of Loop (for Arrays)

let arr = [1, 2, 3, 4, 5];
for (let num of arr) {
  console.log(num);
}

Arrays

Arrays are used to store multiple values in a single variable.

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Outputs: apple

Objects

Objects store collections of data in key-value pairs.

let person = {
  name: "John",
  age: 30,
  greet: function() {
    return "Hello, " + this.name;
  }
};

console.log(person.name); // Outputs: John

Array Methods

Commonly used methods for working with arrays:

  • push(): Adds an item to the end of an array. Example: fruits.push("orange");
  • pop(): Removes the last item from an array. Example: fruits.pop();
  • shift(): Removes the first item from an array. Example: fruits.shift();
  • unshift(): Adds an item to the beginning of an array. Example: fruits.unshift("kiwi");
  • map(): Creates a new array by calling a function on each element. Example: arr.map(x => x * 2);
  • filter(): Creates a new array with elements that pass a test. Example: arr.filter(x => x > 10);

Events

JavaScript can respond to user actions through events like clicks or key presses.

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
  alert("Button was clicked!");
});

ES6 Features

JavaScript ES6 (ECMAScript 2015) introduced several new features:

1. Arrow Functions

const add = (a, b) => a + b;
console.log(add(2, 3)); // Outputs: 5

2. Template Literals

const name = "John";
const message = `Hello, ${name}!`;
console.log(message); // Outputs: Hello, John!

3. Destructuring

const person = { name: "John", age: 30 };
const { name, age } = person;
console.log(name, age); // Outputs: John 30

4. Spread Operator

const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];
console.log(arr2); // Outputs: [1, 2, 3, 4, 5]

Conclusion

This cheat sheet covers the basics of JavaScript syntax, including variables, data types, functions, loops, arrays, objects, events, and ES6 features. JavaScript is a powerful language for creating interactive web applications. For more advanced topics, check out resources on promises, async/await, and JavaScript frameworks!
